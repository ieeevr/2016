Workshop Program

1:45pm - 3:35pm - Main Session

  • 1:45pm - Refined experiment of the Altered Human Joystick for User Cohabitation in Multi-stereocopic Immersive CVEs
    • Weiya Chen, Nicolas Ladeveze, Céline Clavel and Patrick Bourdot
  • 2:10pm - Vishnu: Virtual Immersive Support for HelpiNg Users - An Interaction Paradigm for Collaborative Remote Guiding in Mixed Reality
    • Morgan Le Chénéchal, Thierry Duval, Valérie Gouranton, Jérôme Royan and Bruno Arnaldi
  • 2:30pm - 3DEvent: A Framework Using Event-Sourcing Approach For 3D Web-Based Collaborative Design in P2P

  • 2:50pm - 3D Collaborative Interaction for Aerospace industry
    • Damien Clergeaud, François Guillaume and Pascal Guitton
  • 3:05pm - Bringing real world objects, spaces, actions, and interactions into social VR
    • Misha Sra and Chris Schmandt
  • 3:20pm - Relationship controllability – A design of relationship control-available space in CVE
    • Naoko Hayashida

4:00pm - 5:15pm - Special Session about the 3DUI contest

  • 4:00pm - When the Giant meets the Ant - An Asymmetric Approach for Collaborative and Concurrent Object Manipulation in a Multi-Scale Environment
    • Morgan Le Chénéchal, Jérémy Lacoche, Jérôme Royan, Thierry Duval, Valérie Gouranton and Bruno Arnaldi
  • 4:15pm - Collaborative 3D Manipulation using Mobile Phones
    • Jerônimo G. Grandi, Iago Berndty, Henrique G. Debarbaz, Luciana Nedel and Anderson Maciel
  • 4:30pm - Ray, Camera, Action! A Technique for Collaborative 3D Manipulation
    • Wallace Lages
  • 4:45pm - Batmen – Hybrid Collaborative Object Manipulation Using Mobile Devices
    • Marcio Cabral, Gabriel Roque, Mario Nagamura Andre Montes, Eduardo Zilles Borba, Celso Kurashima and Marcelo Zuffo
  • 5:00pm - Collaborative Hybrid Virtual Environment
    • Leonardo Pavanatto Soares, Thomas Volpato de Oliveira, Vicenzo Abichequer Sangalli, Márcio Sarroglia Pinho, Regis Kopper
